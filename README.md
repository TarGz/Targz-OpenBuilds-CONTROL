# Targz Pen Plotter Ctrl

CNC / Pen Plotter host interface based on OpenBuilds CONTROL (Grbl).

## Install

```
npm install
```

## Run locally (dev)

```
npm run run-local
```

## Build macOS app (arm64)

```
npm run build-mac
```

Output: `dist/TargzPenPlotterCtrl-<version>-arm64.dmg`

Open the DMG, drag to `/Applications`, right-click > Open on first launch (unsigned).
